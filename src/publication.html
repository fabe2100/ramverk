Publikation



För att publicera min sida så valde jag Netlify. Denna plattform som är dansk amerikansk skapades 2014.
Man kan bygga och lägga webpplatsen direkt via Git. Detta görs lätt genom att efter ha skapat ett konto hos Netlify så kan man koppla ihop sitt Github konto direkt via sidan för att sedan välja dem filerna man skulle vilja publicera.
För den som har just startat så kan man göra allt detta gratis men om däremot väljer att ha liksom mer hastighet eller att ha ett säkrare hemsida med att ha ett SLA kontrakt som skyddar båda klienten och leverantören.
Netlify stödjer funktioner som är bildade med Typescript, javascript eller Go. Går att köra med de olika ramverken(React, Vue, Next...).Netlify kan också ansluta till nästan vilken APi som helst.
Jag började först med att skapa ett konto hos Netlify med mitt email, Där på sidan man kommer in valde jag att addera en ny sida. Jag valde då ett existerarnde projekt.
Där anslöt jag till mitt Github konto för att välja filerna jag redan hade puschat från visual studio code till den nya repositorin jag skapat.
Man fick då välja som t.ex att välja vilket bransch etc... Efteråt kunde man klicka på deploy site för att publicera hemsidan på netlify.
