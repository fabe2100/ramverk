Bootstrap




Bootstrap är ett front-end ramverk för att hanetra CSS på ett smidigare sätt. Den innehåller färdiga designmallar för knappar, formulär, navigering, tabeller med mera.
Detta ramverk ger också möjligheten att skapa design som är responisvt för att anpassa de andra skärmstorlekar.
På denna undersida så använder jag Bootstrap 5 som är den senaste versionen som kom ut 2021. Skillnaden mellan Bootstrap 4 ,3,2 och 5 är den nya versionen använder Javascript istället för JQuery.

När jag experimenterade med Bootsrap och Tailwind så insåg jag att Bootsrap är lite begränsad i jämförelse med Tailwind CSS men i samma tid lite enklare än Tailwind CSS.
Det är också lättare att jobba med responsivitet med hjälp av Bootstrap som anpassar designerna lättare.
Jag började använda Bootstrap med hjälp av CDN länken jag kopierat från hemsidan av Bootstrap på:

https://getbootstrap.com/
